First-time setup — 2 minutes:
  1. Go to jsonbin.io → sign up free → click Create Bin
  2. Paste in [] as the initial content and save
  3. Copy your Bin ID (the string in the URL after /b/) and paste it below
  4. Go to API Keys in your JSONBin dashboard → copy your Master Key or create an access key with write permission → paste it below
  5. Open your portfolio HTML → find YOUR_BIN_ID_HERE → replace it with the same Bin ID
Your secret key is stored only in this browser's localStorage — it never touches your portfolio file or any server you don't control.
